=encoding utf8

=head1 NAME

B<mario> - a plumber

=head1 SYNOPSIS

B<mario> [I<options>] I<MSG> {raw, url}

=head1 DESCRIPTION

B<mario> is a powerful plumber.

=head1 OPTIONS

=over 4

=item B<--config> I<FILE>

Configuration file to use.

=item B<--guess>

Guess the kind of the message.

=item B<-h, --help>

Display the help message and quit.

=item B<--rule> I<FILE>

Rules file to use.

=item B<-v, --verbose>

Increase the configured verbosity level  by  one. Specify multiple times to
increase log level multiple times.

=back

=head1 EXIT STATUS

B<mario> exits 0 if a rule successfully matches and all actions run successfully
execute, and >0 if an error occurs.
